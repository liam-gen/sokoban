## A propos

Ce dépôt contient le jeu Sokoban développé dans le cadre de la SAE 1.01 - Implémentation d'un besoin client, projet du cursus BUT Informatique de l'IUT de Lannion. Il s'agit d'un jeu où l'on doit déplacer des caisses sur des cibles. Ce jeu dispose de 2 versions.
Langage utilisé : **C**



### Versions

Voici une liste des versions et des fonctionnalités demandées.

#### Version 1
- Jeu basique (déplacements, ...)
- Sauvegarde de partie

#### Version 2
- Sauvegarde des déplacements
- Possibilité de revenir à l'action précédente
- Possibilité de zoomer sur le plateau

   ```sh
   git clone https://github.com/liam-gen/sokoban.git
   ```
3. Compiler le jeu
   ```sh
   make build
   ```
4. Lancer le jeu
   ```sh
   make run
   ```
5. Vérifier les conventions de codage (optionnel)
   ```sh
   make verif
   ```
